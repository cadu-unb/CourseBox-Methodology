## Comandos úties

### Na raiz do projeto

#### Novos Pacotes

- Para adicionar pacotes da estrutura do projeto:

```cmd
uv add {nome_do_pacote}
```

#### Tree

- Para criar uma arvore de arquivos para nos ajudar a descrever e debugar o código:

```cmd
uv run python 2_devtools/py/file_tree.py
```
